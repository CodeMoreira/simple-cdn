Introduction & Authentication

The registry uses token-based Bearer authentication for programmatic access (such as deploy pipelines or external scripts). Using the browser, your active session manages access automatically, but any automated workflow MUST transmit an Authorization header.

Authorization: Bearer <YOUR_API_TOKEN>

You can generate or retrieve your personal API Token from the My Profile section. Keep it secure and do not commit it to version control.
